About the project

LaceUp is a meetup app designed to connect people who want to play sports together. Users can find other players of the same skill level to play against for just the right amount of challenge. Users can create events, or join already existing events specific by park and event type. When joining an event, users are able to see the number of teams required for the event, and with the click of one button, lace up for their team of choice!

The site is built using a MERN stack. I worked as the frontend lead for this project and managed the combined effort between myself and three other developers. Our frontend utilizes React-Redux across the board. Three features I authored in terms of frontend are user auth, search functionality, and our event explore page.

Above is an example of the search and explore page which uses the Google Maps API as a centralized location to view all events happening in a specified location. From this view, users may select a park and view all available events in a side window.
